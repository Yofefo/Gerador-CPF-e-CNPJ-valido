# Funções Disponíveis

- **generate_cnpj**: Geração de CNPJ.
- **generate_cpf**: Geração de CPF.

## Uso

Aqui está um exemplo de como usar as funções neste pacote:

```python
from cpf_and_cnpj_generator import generate_cnpj, generate_cpf
```

```python
# Gerar um CNPJ válido
cnpj = generate_cnpj()
print(cnpj)

# Gerar um CPF válido
cpf = generate_cpf()
print(cpf)
```
